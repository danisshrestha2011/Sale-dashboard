# Smart Personal Finance Tracker

A lightweight, browser-based personal finance dashboard for tracking income, expenses, and savings. This project is built with HTML5, Tailwind CSS, and Chart.js, featuring data persistence using `localStorage`.

## üöÄ Features

-   **Dashboard Overview**: Real-time metrics for Total Balance, Monthly Income, Monthly Expenses, and Savings Rate.
-   **Interactive Visualizations**:
    -   **Cash Flow Overview**: A line chart showing balance history over time.
    -   **Spending by Category**: A doughnut chart breaking down expenses by category.
-   **Transaction Management**:
    -   Add new transactions with date, description, category, and amount.
    -   Automatically categorizes as Income or Expense.
    -   Delete transactions from the history.
-   **Data Persistence**: All data is saved in your browser's `localStorage`, so your information stays even after refreshing the page.
-   **Responsive Design**: Fully responsive UI built with Tailwind CSS.
